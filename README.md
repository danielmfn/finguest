# Readme

### Requisitos

Requisitos necessários para rodar o projeto:

* JDK 17.0.6
* Maven 3.8.4

### No diretório do projeto executar os seguintes comandos:
* mvn clean install -U
* mvn spring-boot:run

### Link Swagger
* url: http://localhost:8081/swagger-ui.html

### H2 Database
Acesso ao painel do H2 Database:

* url: http://localhost:8081/h2
* usuário: sa
* senha: admin
